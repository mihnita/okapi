Okapi Library for XLIFF 2.

The goal of this library is to provide Java developers with a simple way to read or create XLIFF 2 documents. For example, it provides:

  • A set of classes to store and manipulate the parsed <unit> elements of an XLIFF document.
  • A class to read XLIFF documents using a stream reader (allowing for very large documents).
  • A class to create XLIFF documents.
